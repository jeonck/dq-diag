# 데이터베이스 품질 진단 툴

공공데이터 품질관리 매뉴얼을 기반으로 한 데이터베이스 품질 진단 도구입니다.

## 주요 기능

### 진단 지표

1. **완전성 (Completeness)**
   - 기본키(PK) 존재 여부
   - NULL 값 검사
   - 미사용 컬럼 검사
   - 데이터 타입 일치성

2. **일관성 (Consistency)**
   - 컬럼명 명명 규칙
   - 데이터 타입 일관성
   - 중복 데이터 검사
   - 코드값 일관성
   - 날짜 형식 일관성

3. **정확성 (Accuracy)**
   - 도메인 정확성 (여부, 코드 등)
   - 범위 정확성 (음수, 비율, 나이 등)
   - 형식 정확성 (이메일, 전화번호 등)
   - 날짜 유효성
   - 논리적 일관성 (시작일 < 종료일)

4. **보안성 (Security)**
   - 개인정보 노출 검사
   - 민감정보 검사
   - 암호화 필요 데이터 검사

5. **적시성 (Timeliness)**
   - 데이터 최신성 검사
   - 갱신 주기 검사
   - 미래 날짜 검사

6. **유용성 (Usability)**
   - 데이터 충분성
   - 컬럼 유용성
   - 데이터 다양성

## 설치 방법

### 1. 의존성 설치

```bash
pip install -r requirements.txt
```

### 2. 실행

```bash
streamlit run app.py
```

## 사용 방법

### 1. 데이터 업로드
- 왼쪽 사이드바에서 CSV 파일을 업로드하거나
- "샘플 데이터 사용" 버튼을 클릭하여 샘플 데이터 사용

### 2. 진단 지표 선택
- 원하는 품질 진단 지표를 선택합니다
- 기본적으로 완전성, 일관성, 정확성이 선택되어 있습니다

### 3. 진단 실행
- "진단 시작" 버튼을 클릭하여 품질 진단을 실행합니다

### 4. 결과 확인
- 전체 품질 점수 및 등급을 확인합니다
- 지표별 상세 진단 결과를 확인합니다
- 발견된 이슈 및 개선 방안을 확인합니다

### 5. 리포트 다운로드
- 진단 결과를 JSON 형식으로 다운로드할 수 있습니다

## 샘플 데이터

`sample_data/` 디렉토리에 다음 샘플 데이터가 포함되어 있습니다:

1. **sample_customer.csv**: 고객 데이터
   - 개인정보, 날짜, 코드값 등 다양한 데이터 타입 포함
   - 의도적으로 품질 이슈를 포함하여 진단 테스트 가능

2. **sample_product.csv**: 상품 데이터
   - 재고, 가격, 할인율 등 범위 검증이 필요한 데이터 포함
   - 일관성 및 정확성 테스트에 적합

### 샘플 데이터에 포함된 품질 이슈 예시

**완전성 이슈**:
- NULL 값이 포함된 컬럼
- 중복 레코드 (마지막 행)

**일관성 이슈**:
- 대소문자가 혼용된 코드값 (Y/y, GOLD/gold)
- 다양한 날짜 형식

**정확성 이슈**:
- 음수 재고 수량
- 비정상적인 나이 값 (150세, -5세, 250세)
- 100을 초과하는 할인율
- 잘못된 전화번호 형식
- 카드번호 형식 오류

**보안성 이슈**:
- 개인정보(이메일, 전화번호, 카드번호) 평문 저장

**적시성 이슈**:
- 오래된 데이터 (2020년 데이터)
- 미래 날짜 (2030년)

## 진단 점수 체계

- **90점 이상**: 🟢 우수 - 높은 품질의 데이터
- **70-89점**: 🟡 양호 - 양호한 품질, 일부 개선 필요
- **50-69점**: 🟠 보통 - 여러 개선사항 존재
- **50점 미만**: 🔴 미흡 - 즉각적인 개선 필요
